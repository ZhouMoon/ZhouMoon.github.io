Windows杀死java进程

2022-05-14 Java, Windows 0 0 评论

查询端口

  • windows 系统查询8080端口的进程
  • netstat -ano | findstr 8080

查询运行的程序

  • tasklist或jps 命令查看当前系统中的进程列表
  • 如查看chrome 进程: tasklist | findstr chrome

杀死进程

  • 通过name: taskkill /im nginx.exe /f
  • 通过pid: taskkill /pid 8568 /f
